Before running PickAndPlaceScenario or Y1MultiplePickAndPlaceScenario
scenarios you will need to run the Y1DemoSimulationScenario from the SimulationScenarios package. That's the simulation scenario with the Y1 scene.

The SimulationScenarios package repository is:

https://github.com/SecondHands/SimulationScenarios.git
